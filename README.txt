########## Français ##########

Cette application est un visualiseur de données acquises via la pateforme de motion capture AmigoCap.
Elle ne requiert le lancement d'aucun serveur web pour pouvoir correctement fonctionner.
Pour adapter l'affichage vous pouvez modifier les variables contenues dans le fichier lib/parameters.js.

|-- css                  // Dossier contenant les feuilles de style CSS
|-- Documentation        // Dossier contenant la documentation réalisée dans la cadre de notre projet
|-- Donnees              // Dossier contenant divers jeux d'essais capturés à l'aide de la plateforme
  |-- benjamin           // Capture d'un humain
  |-- box                // Capture d'une boite immobile
  |-- movedBox           // Capture d'une boite en mouvement
|-- img                  // Dossier contenant les différents images utilisées
|-- js                   // Dossier contenant les scripts JavaScript
  |-- lib                // Dossier embarquant toutes les librairies utilisées
  |-- init.js            // Fichier contenant le code nécéssaire à l'éxecution
  |-- parameters.js      // Fichier contenant toutes les variables
|-- index.html           // Fichier HTML à lancer
|-- README.txt           // Le présent fichier

1) Lancer le fichier index.html avec votre navigateur.
2) Dans "Points" charger le fichier .csv (avant le fichier .ske).
3) A tout moment dans "Skeleton" vous pouvez chargez le fichier .ske (facultatif).
4) Utiliser les options du lecteur pour lancer/stopper, avancer/reculer et ralentir/accélérer.



########## English ##########

This application is a visualizer of data acquired with the motion capture platform AmigoCap.
It doesn't require the running of any web server to correctly work.
For adapt the display you can edit the variables contained in the lib/parameters.js file.

|-- css                  // Folder containing all the Cascading Style Sheets files
|-- Documentation        // Folder containing all the written documentation for our project
|-- Donnees              // Folder containing data sets captured with the platform
  |-- benjamin           // Capture of a human
  |-- box                // Capture of a motionless box
  |-- movedBox           // Capture of a moving box
|-- img                  // Folder containing the used pictures
|-- js                   // Folder containing the JavaScript scripts
  |-- lib                // Folder containing the used libraries
  |-- init.js            // File containing the necessary code for execution
  |-- parameters.js      // File containing all the variables
|-- index.html           // HTML file to run
|-- README.txt           // The present file

1) Run the index.html with your browser.
2) In "Points" load the .csv file (before the .ske file).
3) At any time, in "Skeleton" you can load the .ske file (optional).
4) Use the reader's options to start/stop, forward/bacward and slow down/speed up.
